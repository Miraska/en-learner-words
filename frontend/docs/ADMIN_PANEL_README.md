# Админ панель WordCraft

## Доступ к админке

Админ панель доступна по замудренной ссылке:
```
http://localhost:3000/admin-panel-x7k9m2p8q1w4e6r3t5y7u9i0o1p2a3s4d5f6g7h8j9k0l1z2x3c4v5b6n7m8
```

## Настройка

1. Добавьте в файл `.env` в папке `backend/` следующие переменные:
```env
ADMIN_USERNAME="admin"
ADMIN_PASSWORD="super_secret_admin_password_2024"
```

2. Перезапустите backend сервер

## Функции админки

### Общая статистика
- Общее количество пользователей
- Новые пользователи (сегодня, за неделю, за месяц)
- Статистика словарей (публичные/приватные)
- Статистика слов и изученных слов
- Статистика сессий и подсказок

### Детальная аналитика
- Топ пользователи по активности
- Популярные словари по лайкам
- Статистика по языкам
- Активность за последние 30 дней

### Безопасность
- Доступ только по Basic Auth
- Замудренная ссылка для скрытия админки
- Проверка учетных данных на backend

## Структура данных

Админка показывает следующие метрики:

### Пользователи
- Общее количество
- Новые регистрации
- Статистика подписок
- Топ активных пользователей

### Контент
- Словари (публичные/приватные)
- Слова в базе
- Изученные слова
- Статистика по языкам

### Активность
- Сессии обучения
- Использование подсказок
- Ежедневная активность
- Популярные словари

## Безопасность

- Ссылка на админку замудрена для предотвращения случайного доступа
- Требуется Basic Authentication
- Учетные данные хранятся в переменных окружения
- Все запросы проходят через middleware проверки прав
