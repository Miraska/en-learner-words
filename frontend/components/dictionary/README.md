# Компоненты словарей

Этот модуль содержит компоненты для работы с публичными словарями.

## Компоненты

### DictionaryCard

Карточка словаря с информацией о названии, описании, лайках, языках и количестве слов.

**Props:**

-   `dictionary` - объект словаря с полями:
    -   `id` - ID словаря
    -   `name` - название словаря
    -   `description` - описание (опционально)
    -   `likes` - количество лайков
    -   `sourceLanguage` - исходный язык
    -   `targetLanguage` - целевой язык
    -   `createdBy` - информация о создателе
    -   `_count.words` - количество слов

**Функциональность:**

-   Отображение информации о словаре
-   Возможность лайкать словарь
-   Ссылка на изучение словаря

### DictionaryFilters

Компонент фильтров для поиска и сортировки словарей.

**Props:**

-   `onFiltersChange` - функция обратного вызова при изменении фильтров

**Функциональность:**

-   Поиск по названию словаря
-   Фильтрация по исходному языку
-   Фильтрация по целевому языку
-   Сортировка по лайкам, названию или дате создания
-   Выбор порядка сортировки (по возрастанию/убыванию)

### DictionaryStats

Компонент для отображения статистики словаря.

**Props:**

-   `wordCount` - количество слов
-   `author` - автор словаря
-   `createdAt` - дата создания (опционально)

### LanguageDisplay

Компонент для отображения языков словаря.

**Props:**

-   `sourceLanguage` - исходный язык
-   `targetLanguage` - целевой язык

## Использование

```tsx
import DictionaryCard from './DictionaryCard';
import DictionaryFilters from './DictionaryFilters';

// В компоненте
<DictionaryFilters onFiltersChange={handleFiltersChange} />
<DictionaryCard dictionary={dictionary} />
```
